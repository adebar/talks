Setup (2/2)

  1. Open Port 1234 on your EC2 instance.

  2. Run docker image:

    docker run -it --rm \

    -v "/home/ec2-user/notebooks":"/home/jovyan" \

    -p 1234:8888 \

    jupyter/scipy-notebook \

    start.sh jupyter lab \

    --NotebookApp.password='sha1:123..321'

  3. Your JupyterLab should now be accessible at
    http://ec2-12-34-45-67.compute-1.amazonaws.com:1234/.
